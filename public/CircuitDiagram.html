RPi 4B Sensor Circuit

DS18B20 Temperature · MQ2 Gas Detection · Firebase IoT

CIRCUIT DIAGRAM
⚠️ The 4.7kΩ resistor is REQUIRED for DS18B20. Connect it between the DATA pin and VCC (3.3V). Without it, the sensor will not be detected. MQ2 AO pin — do NOT connect, only use DO pin.
DS18B20 — TEMPERATURE SENSOR
PIN 1 GND → RPi Pin 6 (GND)
PIN 2 DATA → RPi Pin 7 (GPIO4)
PIN 3 VCC → RPi Pin 1 (3.3V)
RESISTOR 4.7kΩ DATA ↔ VCC
MQ2 — GAS SENSOR
VCC 5V → RPi Pin 2 (5V)
GND GND → RPi Pin 6 (GND)
DO GPIO17 → RPi Pin 11 (GPIO17)
AO N/C ← Do NOT connect
SETUP PROCEDURE
01
Enable 1-Wire Interface
Add dtoverlay=w1-gpio,gpiopin=4 to /boot/firmware/config.txt then reboot. Required for DS18B20 communication.
02
Verify DS18B20 Detection
Run ls /sys/bus/w1/devices/ — you should see a device starting with 28- which is your sensor's unique ID.
03
Adjust MQ2 Sensitivity
Use a small screwdriver to turn the blue potentiometer on the MQ2 board. The onboard LED triggers when gas exceeds threshold. Set in clean air first.
04
Run the Logger
Execute python main.py from the project directory. Data overwrites Firebase every 2 seconds at the /sensor_data node.
